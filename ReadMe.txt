The project is an html page (client side only) that displays information about cryptocurrencies while synchronizing between different sources of information.

Main challenges:

Queries from APIs using Ajax
Synchronize the information obtained from different APIs
Coordination between "brother" divs and using event listeners
Bootstrap design 
Parallax scrolling
